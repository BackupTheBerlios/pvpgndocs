Introduction

PvPGN Banners are displayed at the top of the screen when chatting on Battle.net (TM) or a PvPGN server. The real Battle.net (TM) uses these for commercial advertisements, but you can use them for whatever you want!

By default, there are ads for PvPGN displayed here which link to the PvPGN website, but by following the instructions below, you can easily create your own banners to display in this space.

Unfortunately, Blizzard uses some very strange image formats, which makes this not as easy as it sounds.

Graphic Formats

The formats that are supported are PCX, MNG, and SMK. These are the only known formats that are supported by the Blizzard clients. For animated banners, you must use the SMK (Smacker) format or MNG format. The actual size of the images MUST be 468x60 pixels or the image will not display

  • *.mng only works for Warcraft III
  • *.pcx reportedly works for Starcraft, though we haven't got one to work yet
  • *.smk is known to work for Starcraft and Diablo.

There is however a catch, there's always a catch. PvPGN does not ship with "*.pcx" files. A "*.pcx" banner displayed in StarCraft will look like a random selection of colours unless you restrict your colour selection to 64 colours starting at colour number 32. How do you do that to a PCX file? We don't know.

Most graphics programs can create a *.pcx file, if you're looking for some freeware to do this, use The Gimp which is available for all platforms: http://www.gimp.org

"*.smk" files are considered to be better than the other formats by the developers and supported by all clients. They don't have palette limitations and they compress better. Though just to add to the effort required here, no normal graphics program can produce a "*.smk" file.

For Diablo it appears as though the smk file will need to contain multiple frames (210 for example and all the same image). The good news here is that it compresses really well.

So your best output format is "*.smk" but how do you produce it?
You need "SmackerTools":
- It's free! (Phew!)
- It only runs on Windows (ugg!)
- It is really important to use the OLD version of SmackerTools

Very important. The current version of RadTools for generating Smacker files will only generate smacker files that will display in a Mac.

For full functionality, you must use an older version of the RadTools. Download it from

http://www.radgametools.com/smkdown.htm, look for "The Old Smacker Tools"

Download SmackerTools from:
http://www.radgametools.com/down/Smacker/SmkTools.exe

SmackerTools can take many different filetypes as input. Try it and see what works best for you. Also have a read of the SmackerTools FAQ on their web site.

Preparation

  1. Get hold of a graphical editor, "Paint Shop Pro", "Photoshop", almost anything. I have used Paint Shop Pro (http://www.jasc.com/index.html). It has excellent support for manipulating color pallets (save, load, etc) on Windows.

    Jasc Paint Shop Pro 8 also includes an Animation Editor. You access it from the File / Jasac Software Products menu. Select Jasac Anamination Shop. This is the only program I know at the moment that will generate "*.mng" files. Remember you need to create "*.mng" files for Warcraft.

    How do you generate *.mng files under Mac OS X, any know? Please help.

  2. Find your ad.conf file. This will be in your "etc" directory (Linux) or your "conf" directory (Windows). Read it carefully. Notice that you have ad000001.smk and ad000001.mng (one will display on StarCraft) and the other exclusively for Warcraft)

  3. READ THE HELP TEXT at the beginning of the ad.conf file!

Smack your banner into shape

From the main screen of the RAD Video Tools, highlight the animation that you want to convert, and click the "SmackIt" button at the bottom of the window. Once the RAD Compressor window opens, you'll be able to tell SmackerTools what colour range to use.

Creating a banner with a single frame

  • Enter 64 (even though I've used 63 in the example) in the "To Use:" field of the Palette area on the screen
  • Enter 32 (even though I've used 33 in the example) in the "To begin at:" field
  • Also make sure that "create ring" is checked
  • Click the Smack button to convert your graphic.

Take a look at these screenshots to help you:





You have the option of converting a single image into a single converted file or converting multiple files into a single smacker file. I'll leave you to figure out how to do that.

 

Creating your banner with multiple frames

  1. To create smk files with a number of frames (all based on the same image) do the following:
  2. Create a "*.lst" file for smacker, so we'll call it advert.lst
  3. Inside this file (it's just text) we'll provide a file name and some parameters, e.g. "E:\pvpgn\files\ad000001.psd /L210". All this information goes on one line. You can have multiple entries on successive lines that will combine together to create one smacker file.
  4. Back in SmackerTools, select the graphic processor tab, ensure that you filter the list of files on the left so that it just lists ".lst" files. Select your list file and click convert. SmackerTools will do its stuff. It may take 30 seconds, it may take 10 minutes. The duration depends on the complexity of your graphics and how many entries are in your list. The output from the convert process is *.flc (refer screen shot)

Convert the *.flc file into a *.smk file. (refer screen shot) and you're done.

Oh, want to add sound? You need some WAV files. Then use the Mixer tab and add the sound.

Add the new banner to your server

  1. Place your graphic file in the "files" directory on your server, make sure you follow the file naming conventions.

  2. Edit the ad.conf file making sure your follow the format. See the comments at the beginning of ad.conf to help you. The text below is an example only!
    "ad000001.smk"  init    30 "http://www.yourwebsite.org"      0x000002  "SEXP"
"ad000002.smk"  start   30 "http://www.yourwebsite.org"      0x000003  "SEXP"
"ad000003.smk"  norm    30 "http://www.yourwebsite.org"      0x000002  "SEXP"
    Special note for WarCraft entries
    Entries in ad.conf can only be of type "init" and are randomly chosen, thus if you have more than one banner to display you must have them in separate files and separate entries in the ad.conf file. This might be a bug or by design?

    "*.mng" files with multiple images do work but seem to have a timing limitation that cannot be influenced by the timer in ad.conf

  3. Note the last column. This column will restrict display of this new advertisement to the specific client gametag. So in this case Starcraft:Broodwar. Setting this to "NULL" will show this banner to all clients that DON'T already have a client-specific banner

Notes that don't fit anywhere else

You can save banners from Battle.net (TM) by using bnftp. bnftp is included in the PvPGN source tarball. You will need to know the exact filename.

Sound encoding is possible with the SMK format. As to if the clients will play the sound, well you experiment and tell us.
