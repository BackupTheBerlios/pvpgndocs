Introduction

Using a tracker is a way of getting your server noticed. You enhance your server configuration to notify a tracking service of your existence. At intervals that you specify, your server will continue to notify the tracking service of your existence. The tracking service usually publishes details of your existence on a web page. For example, have a look here: http://pvpgn.mivabe.nl/index.php.

Substancial portions of this documentation have been copied from: http://pvpgn.mivabe.nl/howto/tracker.php.

On with the show. Here's how you configure your server.

 

Tracker Set Up

Open your bnetd.conf file, and change the line's acording to this

track = 180
trackaddr = "bnet.mivabe.nl"


Save it, then go in your server an restart it (/rehash would do as well)
.

Final Touch

Rember when you change somting, the bnet.mivabe.nl tracker only update's efry 15 mins, so you problerbly wont see it right away just wait a few minutes

A full / complete example of a server set is as follows:

track = 180
trackaddrs = "bnet.mivabe.nl"
location = ":bs Earth"
description = "SCSBW 1.11b OPE - xxx"
url = "http://www.yourplace.org"
contact_name = "dgwilson65 & U-238 "
contact_email = "someonespecial@nospam.org"
